### Project Description
Office document generation for Genropy - Word and Excel builders using python-docx and openpyxl.

### Key Components

- **WordBuilder**: Generates Word documents (.docx) using python-docx
- **ExcelBuilder**: Generates Excel spreadsheets (.xlsx) using openpyxl

### Dependencies

Optional dependencies (installed separately):
- `python-docx>=1.0.0` for Word support
- `openpyxl>=3.1.0` for Excel support
