# Advanced Word Document

Demonstrates all advanced features of the WordBuilder.

## Features

- **Document settings**: orientation, margins
- **Text formatting**: bold, italic, underline, colors, fonts
- **Inline runs**: mixed formatting within a paragraph
- **Lists**: bulleted and numbered
- **Tables**: formatted with colors, alignment, column widths
- **Headers and footers**

## Run

```bash
python main.py
```

Creates `output.docx`.
