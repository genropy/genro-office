# Advanced Excel Spreadsheet

Demonstrates all advanced features of the ExcelBuilder.

## Features

- **Cell formatting**: colors, borders, alignment, number formats
- **Merged cells**: horizontal and vertical merges
- **Freeze panes**: keep headers visible while scrolling
- **Autofilter**: filterable data ranges
- **Charts**: bar, line, and pie charts

## Run

```bash
python main.py
```

Creates `output.xlsx`.
